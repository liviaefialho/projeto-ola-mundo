Maldição do Olá Mundo

Entre os programadores existe uma ‘tradição’ que o primeiro programa a ser feito, ao aprender uma nova linguagem deve ser o “Olá Mundo!” ou “Hello World” (em inglês), ou seja, fazer mostrar na tela para o usuário essa mensagem…


O “Olá, mundo” é um pequeno código que pode ser desenvolvido em diferentes linguagens de programação. Seu objetivo é facilitar o conhecimento da estrutura básica e o funcionamento da linguagem.


Para quê isso? Bom, reza a lenda que o programador quando vai aprender uma linguagem de programação nova, ele deve em qualquer circunstância, em seu primeiro programa imprimir na tela as palavras da maldição, pois se o mesmo não fizer isso, ele não irá aprender adequadamente essa nova linguagem e não será um bom profissional.


Ele também serve para verificar se um ambiente está preparado adequadamente, pois a instalação de algumas linguagens de programação em um computador pode requerer outras instalações e configurações. Assim, o Hello World também é uma forma de verificar se o ambiente está configurado de maneira adequada. Portanto, se o programa funcionar, é sinal que está tudo certo com a preparação da máquina.
